Html基础

<!--声明文档类型,html对大小写不敏感-->
<!DOCTYPE html>
<!--<起始标签名>要显示的内容</结束标签名>-->
<html>
	<head>
		<!--元数据-->
		<!--声明页面的"编码格式"-->
		<!--单标签-->
		<meta charset="utf-8" />
		<title>标题</title>
		<meta name="viewport" 
			content="
			width=device-width,     //--布局窗口宽度等于设备物理像素 兼容ios，主流安卓机
			inintal-scale=1,        //--初始放大系数 兼容旧的安卓机 
			minimum-scale=1,        //--最小的放大系数 
			maximum-scale=1,         //-- 最大的放大系数 
			user-scalable=no"/>
			移动端开发，响应式
			//< meta name="viewport" 
			content="width=200,inintal-scale=1,minimum-scale=1,maximum-scale=1,             
			user-scalable=no"/>
	</head>
	<body>
		<a href="html/dizhi/相对地址.html">回到相对地址.html</a>
		<!--标签是可以嵌套的-->
		<!--图片类标签-->
		<!--img标签 src 将外部资源引用到当前页面-->
		<!--alt 当资源引用失败的时，显示的文本-->
		<!--title 当你的鼠标悬停在图片上，显示的文本-->
		<!--设置图片宽度 width
		设置图片高度 height
		将下面图片设置宽和高为300px-->
		<img src="https://www.baidu.com/img/bd_logo1.png" alt="https://www.baidu.com/img/bd_logo1.png"
			title="百度logo" width="300" />
		<!--超链接标签 a-->
		<!--target属性 指定链接打开方式
				  _self,_blank-->
		<a href="http://baidu.com" target="_self">点我去百度</a>
		<!--鼠标悬停在文字上时，显示"这是链接"-->
		<a href="http://baidu.com" target="_blank" title="这是链接" >
			<!--标签是可以嵌套的-->
			<img src="https://www.baidu.com/img/bd_logo1.png" alt="https://www.baidu.com/img/bd_logo1.png"
			title="百度logo" width="300" />
			点我去百度
		</a>
	</body>
</html>

表单，下述表单的显示需要实体显示，否则会被浏览器解析

& lt; 代替 < (&后没有空格，现在是分开，合在一起会被解析)
& gt; 代替 > (&后没有空格，现在是分开，合在一起会被解析)
& nbsp; 代替 空格 (&后没有空格，现在是分开，合在一起会被解析)
& reg; 代替 注册商标(®) (&后没有空格，现在是分开，合在一起会被解析)
& copy; 代替 版权(© ) (&后没有空格，现在是分开，合在一起会被解析)
