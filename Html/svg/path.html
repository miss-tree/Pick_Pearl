path属性

  • M：移动到（moveto）x,y
  • L：画直线到（lineto）x,y
  • Z：闭合路径
  • H：从当前坐标画水平直线到 x 坐标轴
  • V：从当前坐标画水平直线到 y 坐标轴
  • C：参数=>x1 y1 x2 y2 x y,画条贝赛尔曲线到 x， y，其中x1 y1 x2 y2为控制点
  • S：参数=> x2 y2 x y,画条贝赛尔曲线到 x， y，其中 x2 y2为控制点
  • Q：参数=> x1 y1 x y,画条贝赛尔曲线到 x， y，其中 x1 y1为控制点
  • T：参数=>x y,画条反射二次贝赛尔曲线到 x， y，其中以前一個坐标为控制点
  • A：参数=>rx ry x-axis-rotation large-arc-flag sweep-flag x y, 从目前点的坐标画個橢圆形到指定点的 x, y 坐标： 其中 rx, ry 為橢圆形的 x 轴及 y 轴的半径，x-axis-rotation 是弧线与 x 轴的旋转角度， large-arc-flag 則設定 1 最大角度的弧线或是 0 最小角度的弧线， sweep-flag 设定方向为 1 顺时针方向或 0 逆时针方向 ( Arc )
