Media Queries: grid

  • 版本：CSS3
  • 是否接受min/max：否

语法：

grid：<integer>

取值：

<integer>：
用整数值来定义是否使用栅格或点阵。只有1和0才是有效值，1代表是，0代表否

说明：

用来查询输出设备是否使用栅格或点阵。
  • 本特性不接受min和max前缀。

  • 简单列举几个应用示例：

    @media all and (grid){ … }
@import url(example.css) all and (grid:0);

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
较早版本 6-8 4 5.1.7 13 11.5
较近版本 9
