behavior

  • 版本：CSS1
  • 媒体：视觉

语法：

behavior：<url> | url(#objID) | url(#default#behaviorName)

适用于：所有元素

继承性：无

动画性：否

计算值：指定值

取值：

<url>：
使用绝对或相对地址指定DHTML行为组件（.htc）
url(#objID)：
用二进制实现DHTML行为，#objID为object对象指定的id特性
url(#default#behaviorName)：
IE的默认行为。由behaviorName指定

说明：

设置或检索对象的DHTML行为。
  • 多个行为之间用空格隔开。
  • 对应的脚本特性为behavior。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
Values IE
Basic Support 6.0-9.0
10.0-11.0

示例：

div{behavior:url(fly.htc) url(shy.htc);}
