box-flex-group

  • 版本：CSS3
  • 媒体：视觉

语法：

box-flex-group：<integer>

默认值：1

适用于：伸缩盒子元素

继承性：无

动画性：否

计算值：指定值

取值：

<integer>：
用整数值来定义伸缩盒对象的子元素所在的组。

说明：

设置或检索伸缩盒对象的子元素的所属组。
  • 动态给数值较大的组分配其内容所需的实际空间（如无内容、padding、border则不占空间），剩余空间则均分给数值最小的那个组（可能有1个或多个元素）
  • 对应的脚本特性为boxFlexGroup。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
