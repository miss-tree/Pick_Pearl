page-break-inside

  • 版本：CSS2
  • 2015.8.12

语法：

page-break-inside：auto | avoid

默认值：auto

适用于：所有元素

继承性：无

动画性：否

计算值：指定值

媒体：页面

取值：

auto：
如果需要，在当前对象内部插入页分割符
avoid：
避免在当前对象内部插入页分割符

说明：

检索或设置对象容器内部出现的页分割符。
  • 对应的脚本特性为pageBreakInside。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
