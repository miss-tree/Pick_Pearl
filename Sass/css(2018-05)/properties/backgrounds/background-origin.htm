background-origin

  • 版本：CSS3
  • 2018.3.14

语法：

background-origin：<box> [ , <box> ]*

<box> = border-box | padding-box | content-box

默认值：padding-box

适用于：所有元素

继承性：无

动画性：否

计算值：指定值

媒　体：视觉

取值：

border-box：
从border区域（含border）开始显示背景图像。
padding-box：
从padding区域（含padding）开始显示背景图像。
content-box：
从content区域开始显示背景图像。

说明：

指定的背景图像计算background-position时的参考原点(位置)。
  • 对应的脚本特性为：backgroundOrigin。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
