ID选择符(E#id)

  • 版本：CSS1

语法：

E#myid { sRules }

说明：

以唯一标识符id属性等于myid的E对象作为选择符。
  • ID选择符(ID Selector)

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
