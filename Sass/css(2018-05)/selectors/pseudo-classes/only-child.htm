E:only-child

  • 版本：CSS3

语法：

E:only-child { sRules }

说明：

匹配父元素仅有的一个子元素E。
  • 要使该属性生效，E元素必须是某个元素的子元素，E的父元素最高是body，即E可以是body的子元素

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
