{{el.name}} {{el.explain}}

::first-line | 选择文本的第一行

	=>在display值为block, inline-block, table-cell 或者 table-caption中有用
		p:first-line {
		  color: lightcoral;
		}

::first-letter | 选择这一行的第一字

	    p::first-letter{
	      color: red;
	      font-size: 2em;
	    }

::selection| 被用户选中高亮的部分

	    div::selection {
	          color: #409EFF;
	    }

:root | 根元素

	=>在声明全局 CSS 变量时 :root 会很有用
	    :root {
	      --main-color: hotpink;
	      --pane-padding: 5px 42px;
	    }

:first-of-type | 选择指定类型的第一个子元素

		.innerDiv p:first-of-type {
		  color: orangered;
		}

:last-of-type | 选择指定类型的最后一个子元素

		.innerDiv p:last-of-type {
		    color: orangered;
		}

:nth-of-type() | 选择指定类型的子元素

		.innerDiv p:nth-of-type(1) {
		    color: orangered;
		}

:nth-last-of-type() | 在列表末尾选择类型的子元素

		.innerDiv p:nth-last-of-type(1) {
		    color: orangered;
		}

:link | 选择一个未访问的超链接

		a:link {
		    color: orangered;
		}

:checked | 选择一个选中的复选框

		input:checked {
		  box-shadow: 0 0 0 3px hotpink;
		}

:not() | 用来匹配不符合一组选择器的元素

		.innerDiv :not(p) {
		    color: lightcoral;
		}
		<div class="innerDiv">
			<p>Paragraph 1</p>
			<p>Paragraph 2</p>
			<div>Div 1</div>
			<p>Paragraph 3</p>
			<div>Div 2</div>
		</div>

:target | 点击触发

当浏览器地址里的hash(地址里#号后面的部分) 和:target伪选择器指定的ID匹配上时， 它的样式就会在这个ID元素上生效。 

		:target {
			color: #000;
		}
		h2:target {
			color: #f00;
		}		
			
		<h2 id="anchor">Anchor 1</h2>
		<h2 id="anchor2">Anchor 2</h2>
		=>在#后面的部分和元素id相同时，设置的样式就会起效，通常和<a>配合使用
