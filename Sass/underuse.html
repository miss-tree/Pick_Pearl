linear-gradient

	<div class="gradient"></div>
		.gradient{
			width: 400px;
			height: 100px;
			background: linear-gradient(to right, white 33%,blue 70%,red);
		}	//渐变从左到右，白色占总体的33%，蓝色占剩下的70%，红色就不用写，默认是最后剩下，若是值超出剩下的就相当于重写蓝色
		.deg{
			background: linear-gradient(66deg ,blue 30% , red );
		}//只支持百分比  表示以时钟6点的方向为0°，顺时针方向拉渐变
	（在ps中演示比较容易懂，拉一个圆形或方形，时钟6点的方向为0°，往中心的方向拉渐变就是其效果一样的）
		.percent{
			background: linear-gradient(blue ,red );
		}类似比例的样式

clip-path

将一个div设置成矩形

	
		div{
			width:200px;
			height:200px;
			background:#061846;
			border-radius:3px;
			clip-path:polygon(
				100% 0,
				100% calc(100% - 15px),
				calc(100% - 15px) 100%,
				0 100%,
				0 0
			)
		}		
		画多边形的方法是从一个多边形的右边顶点坐标开始，按顺时针顺序最后到左边顶点。
		有多少条边就设置多少个坐标，最少是三边行三个坐标点，
