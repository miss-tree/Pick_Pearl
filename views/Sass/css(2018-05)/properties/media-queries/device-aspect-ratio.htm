Media Queries: device-aspect-ratio

  • 版本：CSS3
  • 接受min/max：是

语法：

device-aspect-ratio：<ratio>

取值：

<ratio>：
指定比率

说明：

定义输出设备的屏幕可见宽度与高度的比率。
  • 如常讲的显示器屏幕比率：4/3, 16/9, 16/10
  • 本特性接受min和max前缀，因此可以派生出min-device-aspect-ratio和max-device-aspect-ratio两个媒体特性。

    简单列举几个应用示例：

    @media screen and (device-aspect-ratio:4/3){ … } @import url(example.css) screen and (min-device-aspect-ratio:4/3);

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
