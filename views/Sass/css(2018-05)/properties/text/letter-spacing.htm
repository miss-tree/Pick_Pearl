letter-spacing

  • 版本：CSS1/3
  • 2018.4.8

语法：

letter-spacing：normal | <length>

默认值：normal

适用于：所有元素

继承性：有

动画性：非normal值时

计算值：指定值

相关属性：word-spacing

媒　体：视觉

取值：

normal：
默认间隔。计算值为0
<length>：
用长度值指定字符间隔。可以为负值。

说明：

指定字符之间的额外间隙。
  • 该属性可以将指定的额外间隙添加到每个字符之后，最后一个单词也会被添加。
  • 对应的脚本特性为：letterSpacing。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
