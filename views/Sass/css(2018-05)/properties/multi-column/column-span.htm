column-span

  • 版本：CSS3
  • 媒体：视觉

语法：

column-span：none | all

默认值：none

适用于：除浮动和绝对定位之外的块级元素

继承性：无

动画性：否

计算值：指定值

取值：

none：
不跨列
all：
横跨所有列

说明：

设置或检索对象元素是否横跨所有列。
  • 对应的脚本特性为columnSpan。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
