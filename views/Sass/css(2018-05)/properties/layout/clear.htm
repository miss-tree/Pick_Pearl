clear

  • 版本：CSS1
  • 2018.2.27

语法：

clear：none | left | right | both

默认值：none

适用于：块级元素

继承性：无

动画性：否

计算值：指定值

媒体：视觉

取值：

none：
允许两边都可以有浮动对象
both：
不允许有浮动对象
left：
不允许左边有浮动对象
right：
不允许右边有浮动对象

说明：

定义了一个元素是否可以放置在它之前的浮动元素旁边，或者必须向下移动在新行中放置。请参阅float属性
  • 当一个元素定义了clear值不为none时，可以被用来清除其之前的浮动元素对自身的影响（不同的取值，对应不同方向的浮动）。
  • 对应的脚本特性为clear。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
