border-color

  • 版本：CSS1
  • 2018.3.6

简写属性语法：

border-color：<color>{1,4}

默认值：看每个独立属性

适用于：所有元素

继承性：无

动画性：看每个独立属性

计算值：看每个独立属性

媒　体：视觉

分拆独立属性语法：

border-*-color：<color>

border-*-color = border-top-color,border-right-color,border-bottom-color,border-left-color

默认值：currentColor

适用于：所有元素

继承性：无

动画性：是

计算值：指定值

媒　体：视觉

取值：

<color>：
指定颜色。

说明：

定义元素的边框颜色。
  • border-color如果提供全部四个参数值，将按上、右、下、左的顺序作用于四边；只提供一个，将用于全部的四边；提供两个，第一个用于上、下，第二个用于左、右；提供三个，第一个用于上，第二个用于左、右，第三个用于下。
  • 如果border-width等于0或border-style设置为none | hidden，border-color及其分拆独立属性将失效。

  • 如果 border-color 未被定义，边框颜色将默认取当前最近的文本颜色，即 currentColor。

    举个例子：

    .demo { border: 5px solid; } <div class="demo"> <p class="text">边框色未被定义时，边框色与最近的文本色相同</p> </div>

    这个例子中，只定义了一个 5px 的实线框而未定义颜色，此时，边框色将会取文本色来使用，所以大家将会看到一个黑色的边框

  • 对应的脚本特性分别为：borderColor, borderTopColor, borderRightColor, borderBottomColor, borderLeftColor。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
