判断对象中是否拥有某个属性或方法

in 运算符

	var obj = {name:’jack’};
	alert(‘name’ in  obj);		//true
	alert(‘toString’ in obj);	//true	
	
	可看到无论是name，还是原形链上的toString，都能检测到返回true。

hasOwnProperty 方法

	var obj = {name:’jack’};
	obj.hasOwnProperty('name');	//true
	obj.hasOwnProperty('toString'); //false	
	
	原型链上继承过来的属性无法通过hasOwnProperty检测到，返回false。
