有一座桥，当你身上钱大于30000时， 每过一次桥支付3000元。当钱小于或等于30000时， 每过一次桥支付你身上10%的钱。 已知你有33000元，问过这座桥多少次之后，身上的钱会少于100块?。

	var total = 33000,
		i = 0;
	while(total >= 100) {
		if(total > 30000) {
			total -= 3000;
		} else {
			total *= .9;
		}
		i++;
	}		
	=>能过这座桥56次
