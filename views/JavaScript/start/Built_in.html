eval()

eval函数 学会他,并且远离他!因为它会将值暴露出来，不安全

	var i = 1;	
	eval("i = i + 1");
	eval("alert(i)");	=>2

escape(编码) 和 unescape(解码)

	var url = "http://www.baidu.com/s?name=小米";	
	url = escape(url);	
	alert(url);	=>"http%3A//www.baidu.com/s%3Fname%3D%u5C0F%u7C73"
	alert(unescape(url));=>"http://www.baidu.com/s?name=小米"
