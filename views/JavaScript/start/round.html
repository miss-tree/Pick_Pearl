循环语句

while

	while (变量<=结束值) { 需执行的代码 } 
					
	while (exp) {	//statements;}

do...while

语句 特点:至少执行一次代码,再判断是否进行下一次循环

	do { 需执行的代码 } while (变量<=结束值)				
	var j=0;
	var S=0;
	do{
		j++; 
		S=S+j;
	}while(j<10)
	console.log("S="+S);=>55

for 循环语句

语句 特点:至少执行一次代码,再判断是否进行下一次循环

	for (变量=开始值;变量<=结束值;变量=变量+步进值) { 	需执行的代码 	} 	
				
	for(var count = 1; count < 11; count++){		
		alert(count);	
	}

break、continue

break 语句可以立即退出循环; continue 语句只是退出当前循环;

	var n = 0;	
	for(var i = 1; i < 10; i++){		
		if(i == 4){		    
			continue;		
		}		
		n++;	
	}	
	alert(n);
