base

控制整个页面所有外部链接的根地址会自动拼接在该页面下面的所有相对地址的前面

	<head>
		<base href="/20160727HTML/" />	
	</head>				
	
	<body>		
		<!– -这里的相对地址，会自动拼接base的值，即/20160727HTML/img/xx.jpg-- >
		<img src=“img/xx.jpg”/>	
	</body>

icon

32X32尺寸，并把图片放到项目目录中.在页面head标签中，插入以下代码，href的值为你的icon相对地址

	<head>
		<link rel=“shortcut icon” href=“img/web.icon/>	
	</head>

让你的页面变成一个编辑器

给目标元素添加contenteditable 属性，或设置为true

	<!DOCTYPE html>			
	<html>
		<body>
			<div contenteditable="true">
				This text can be edited by the user. 
			</div>
		</body>
	</html>
